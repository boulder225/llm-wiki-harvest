**Function: `normalize_cookie(raw: str) -> str`**
Per D-02: Accept both bare `sk-ant-...` sessionKey value AND full cookie header string.
- If `raw` contains `sessionKey=`, extract the value after `sessionKey=` up to the next `;` or end of string
- If `raw` starts with `sk-ant-` (bare value), use it directly
- Strip whitespace from the result
- Return the bare sessionKey value (without `sessionKey=` prefix)

**Function: `resolve_cookie(cli_cookie: str | None) -> str`**
Per D-01: Priority order: CLI flag > env var > .env file > interactive prompt.
1. If `cli_cookie` is not None and not empty, return `normalize_cookie(cli_cookie)`
2. Check `os.environ.get("CLAUDE_SESSION_COOKIE")`. If present and not empty, return `normalize_cookie(value)`
3. Try to read `.env` file in current directory (simple line-by-line parsing, look for `CLAUDE_SESSION_COOKIE=...`). If found and not empty, return `normalize_cookie(value)`. Do NOT use python-dotenv -- parse manually with a simple loop:
   ```python
   for line in Path(".env").read_text().splitlines():
       line = line.strip()
       if line.startswith("CLAUDE_SESSION_COOKIE=") and not line.endswith("="):
           value = line.split("=", 1)[1].strip()
           if value:
               return normalize_cookie(value)
   ```
4. If none found, use `click.prompt("Session cookie", hide_input=True)` for interactive masked input. Return `normalize_cookie(result)`.

**Function: `resolve_project_uuid(cli_project: str | None) -> str | None`**
Per D-08, D-09: Priority order: CLI flag > env var > .env file > None (will prompt later).
1. If `cli_project` is not None and not empty, return it
2. Check `os.environ.get("CLAUDE_PROJECT_UUID")`. If present and not empty, return it
3. Try to read `.env` for `CLAUDE_PROJECT_UUID=...` (same simple parsing). If found, return it
4. Return None (will trigger interactive selection in CLI)

**Function: `resolve_org_id(cli_org: str | None) -> str | None`**
Per D-06: Priority order: CLI flag > env var > .env file > None (will discover via API).
1. If `cli_org` is not None and not empty, return it
2. Check `os.environ.get("CLAUDE_ORG_ID")`. If present and not empty, return it
3. Try to read `.env` for `CLAUDE_ORG_ID=...`. If found, return it
4. Return None (will trigger API discovery)

Import only: `os`, `pathlib.Path`, `click`.

**Click group and command:**

```python
@click.group()
@click.option("--cookie", default=None, help="Session cookie (sessionKey value or full cookie header)")
@click.option("--org", default=None, help="Organization UUID (skip org discovery)")
@click.option("--verbose", is_flag=True, help="Show raw HTTP details for debugging")
@click.pass_context
def main(ctx, cookie, org, verbose):
    """Export Claude.ai project conversations and files."""
    ctx.ensure_object(dict)
    ctx.obj["cookie"] = cookie
    ctx.obj["org"] = org
    ctx.obj["verbose"] = verbose
```

**Subcommand: `list-projects`**

```python
@main.command("list-projects")
@click.pass_context
def list_projects_cmd(ctx):
    """List all projects in your Claude.ai account."""
```

Implementation:
1. Resolve cookie via `resolve_cookie(ctx.obj["cookie"])` -- per D-01
2. Create `ClaudeAPIClient(cookie=cookie, verbose=ctx.obj["verbose"])` as context manager
3. **Validate cookie (per D-12, AUTH-03):** Call `client.get_organizations()`. Wrap in try/except:
   - `SessionExpiredError`: Print friendly message per D-10: `"Error: Session cookie is invalid or expired.\nHint: Re-extract from browser DevTools > Application > Cookies > sessionKey"` using `err_console.print` with `style="bold red"`. Exit code 1.
   - `RateLimitError`: Print `"Error: Rate limited by Claude.ai. Try again in a few minutes."`. Exit code 1.
   - `APIError`: Print `"Error: API request failed (HTTP {e.status_code})"`. If verbose, also print `e.response_body`. Exit code 1.
4. **Org selection (per D-04, D-05, D-06, AUTH-02):**
   - First check `resolve_org_id(ctx.obj["org"])`. If non-None, use it directly and set `client.org_id = org_id`.
   - Otherwise use the orgs from `get_organizations()` response.
   - If exactly 1 org: auto-select it (per D-05), print `"Using organization: {org.name}"`.
   - If multiple orgs: display a numbered list with `rich.table.Table` (columns: #, Name, Email) and use `click.prompt("Select organization", type=int)` to let user pick. Validate the selection is in range.
   - Set `client.org_id = selected_org.uuid`
5. **List projects (per PROJ-01):** Call `client.list_projects()`.
6. **Display projects (per D-07):** Create a `rich.table.Table` with columns: #, Name, Created, Description. Populate with project data. Print the table using `console.print(table)`.

|----------|-------------|
| User input -> cookie handling | Cookie from CLI/env/prompt must be normalized safely |
| .env file -> config resolution | .env file is user-controlled local file |
| API response -> project display | Project names displayed in terminal (could contain escape sequences) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-05 | Information Disclosure | config.py | mitigate | Cookie masked in interactive prompt via hide_input=True; cookie value never printed to stdout |
| T-01-06 | Tampering | config.py | accept | .env file is local user-controlled file; no integrity check needed for CLI tool |
| T-01-07 | Spoofing | cli.py | mitigate | Validate cookie against /api/organizations before trusting it; detect 401 early |
| T-01-08 | Denial of Service | cli.py | mitigate | Catch RateLimitError and exit gracefully instead of retrying forever |
</threat_model>
