Create `src/claude_dump/fireflies_markdown.py` — pure transformation, no I/O (mirrors `markdown.py` pattern):

`def format_timestamp(seconds: float) -> str`:
- Convert seconds to `HH:MM:SS` format (e.g., 3661.5 -> "01:01:01")
- If seconds < 3600, use `MM:SS` format (e.g., 125.0 -> "02:05")
- Return "00:00" for 0.0

`def render_transcript(transcript: FirefliesTranscript) -> str`:
- Build YAML frontmatter block:
  ```
  ---
  title: {transcript.title}
  date: {transcript.date}
  duration: {format_timestamp(transcript.duration)}
  transcript_url: {transcript.transcript_url}
  source: fireflies
  ---
  ```
- Heading: `# {transcript.title}`
- Attendees section (if meeting_attendees): `## Attendees\n` then bullet list: `- {attendee.displayName} ({attendee.email})` (skip email parenthetical if email is empty)
- Summary section (if transcript.summary and transcript.summary.overview):
  ```
  ## Summary

  {transcript.summary.overview}
  ```
- Action items section (if transcript.summary and transcript.summary.action_items):
  ```
  ## Action Items

  - [ ] {item}
  ```
  (Render as Markdown checkboxes)
- Keywords (if transcript.summary and transcript.summary.keywords):
  ```
  **Keywords:** keyword1, keyword2, keyword3
  ```
- Transcript section: `## Transcript\n`
  - Group consecutive sentences by same speaker_name to avoid repeating the speaker label on every line
  - For each speaker group:
    ```
    **{speaker_name}** [{start_timestamp}]
    {text of sentence 1}
    {text of sentence 2}
    ```
    Where `start_timestamp` is `format_timestamp(first_sentence.start_time)`
  - Separate speaker groups with a blank line
- If no sentences: render `*No transcript content available.*`

`def make_transcript_filename(transcript: FirefliesTranscript) -> str`:
- Reuse `sanitize_title` from `claude_dump.markdown` (import it)
- Format: `{date}_{sanitized-title}_{id[:8]}.md`
- Date: `transcript.date[:10]` if available, else "0000-00-00"
- ID: `transcript.id[:8]`

```python
@dataclass
class FirefliesExportResult:
    transcripts_exported: int = 0
    transcripts_failed: int = 0
    exported_files: list[str] = field(default_factory=list)
```

`def export_fireflies_transcripts(client: FirefliesClient, output_dir: str | Path) -> FirefliesExportResult`:
- Create output_path from Path(output_dir), mkdir(parents=True, exist_ok=True)
- List all transcripts: `summaries = client.list_all_transcripts()`
- Use Rich Progress (same style as exporter.py: SpinnerColumn, TextColumn, BarColumn, MofNCompleteColumn)
- For each summary item:
  - Update progress description with transcript title
  - Fetch full transcript: `full = client.get_transcript(summary.id)`
  - Render: `markdown = render_transcript(full)`
  - Generate filename: `filename = make_transcript_filename(full)`
  - Write to disk: `(output_path / filename).write_text(markdown, encoding="utf-8")`
  - Increment result.transcripts_exported, append filename to exported_files
  - On `FirefliesAuthError`: stop progress and re-raise (mirrors SessionExpiredError handling)
  - On other exceptions: increment transcripts_failed, continue
- Return result
